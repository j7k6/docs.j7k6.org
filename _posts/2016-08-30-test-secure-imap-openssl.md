title: "Test Secure IMAP with OpenSSL"

```bash
openssl s_client -crlf -connect <$SERVER>:993
```

```
# login
A login <$USER> <$PASSWORD>

# show message count
A status INBOX (messages)

# read message body
A fetch <$MESSAGE_ID> rfc822.text

# read message header
A fetch <$MESSAGE_ID> rfc822.header
```
