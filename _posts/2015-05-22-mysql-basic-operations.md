## Insert
```sql
INSERT INTO `<$TABLE>` (`<$FIELD_X>`, `<$FIELD_Y>`, `<$FIELD_Z>`) VALUES ('<$VALUE_X>', '<$VALUE_Y>', '<$VALUE_Z>');
```

## Update
```sql
UPDATE `<$TABLE>` SET `<$FIELD_X>`='<$VALUE_X>', `<$FIELD_Y>`='<$VALUE_Y>', `<$FIELD_Z>`='<$VALUE_Z>' WHERE `<$FIELD_ID>`=<$X>;
```

## Delete
```sql
DELETE FROM `<$TABLE>` WHERE `<$FIELD_ID>`=<$X>;
```
