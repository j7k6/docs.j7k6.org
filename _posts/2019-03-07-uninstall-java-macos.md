title: "Uninstall Java on macOS"
---

```bash
sudo rm -rf "/Library/Internet Plug-Ins/JavaAppletPlugin.plugin" \
  "/Library/PreferencePanes/JavaControlPanel.prefPane" \
  "$HOME/Library/Application Support/Oracle/Java"
```
